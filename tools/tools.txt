Инструменты для разработки, отладки и обслуживания лабораторного стенда.

В этом разделе находятся:
- Утилиты для калибровки датчиков
- Симуляторы для тестирования алгоритмов управления
- Скрипты для автоматизации развертывания
- Диагностические программы

Инструменты разработаны для упрощения процесса настройки и обслуживания стенда.
